# Gappa - Global Voice Messenger 🎙️

**Gappa** is a professional, real-time voice translation and messaging application inspired by the simplicity of WhatsApp. It allows users to communicate across different languages seamlessly using AI-powered speech recognition and translation.

## 🚀 Live Demo
You can try the live app here: [https://asrock420.github.io/Gappa/](https://asrock420.github.io/Gappa/)

## ✨ Key Features
- **Real-time Voice Translation:** Record your voice in one language and have it translated into your friend's language instantly.
- **WhatsApp Style UI:** A clean, familiar interface with chat bubbles and professional header.
- **Secure Authentication:** Integrated with Firebase Auth for secure Email/Password login.
- **Text-to-Speech (TTS):** Incoming messages are automatically read aloud in the translated language.
- **Bilingual Interface:** Toggle between English and Bengali for the entire app UI.
- **Cloud-Powered:** Uses Google Apps Script and Firebase Realtime Database for high-speed messaging.

## 🛠️ Technology Stack
- **Frontend:** HTML5, CSS3 (WhatsApp Theme), JavaScript (ES6+).
- **Backend API:** Google Apps Script (LanguageApp for translation).
- **Database:** Firebase Realtime Database.
- **Authentication:** Firebase Auth.
- **Web Speech API:** For Speech-to-Text and Text-to-Speech functionality.

## ⚙️ How to Use
1. **Login/Register:** Create an account using your Gmail address.
2. **Select Languages:** Choose your input language (e.g., Bengali) and the recipient's language (e.g., English).
3. **Record:** Hold the microphone icon and speak.
4. **Send:** Preview the translation and hit the Send (➤) button.
5. **Listen:** The recipient will automatically hear the translated message aloud.

## 🛡️ Security
This project uses **API Key Restrictions** and **Firebase Security Rules** to ensure that data is only accessible to authenticated users.
